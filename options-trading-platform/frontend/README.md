# Frontend - Options Trading Automation Platform

This is a basic React frontend for the Options Trading Automation Platform.

## Features
- Home page
- Login and Register forms
- Place Order form
- View Orders

## Setup
1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm start
   ```

## Notes
- Update API endpoints in the components to match your backend URL if needed.
- This is a minimal scaffold. You can extend it with authentication, error handling, and more features as required.
